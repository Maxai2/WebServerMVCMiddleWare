
WEB server


Понятие веб-сервер можно рассматривать с двух сторон: аппаратной и программной.

С аппаратной точки зрения , «веб-сервер» - это компьютер, который хранит файлы сайта 
(HTML-документы, CSS-стили, JavaScript-файлы, картинки и другие) и доставляет их 
на устройство конечного пользователя (веб-браузер и т.д.). Он подключен к сети 
Интернет и может быть доступен через доменное имя, подобное mozilla.org.

С программной точки зрения, веб-сервер включает в себя несколько компонентов, которые 
контролируют доступ веб-пользователей к размещенным на сервере файлам, как минимум 
- это HTTP-сервер. HTTP-сервер - это часть ПО, которая понимает URL’ы (веб-адреса) 
и HTTP (протокол, который ваш браузер использует для просмотра веб-станиц).


Статический веб-сервер состоит из компьютера, на котором установлен HTTP-сервер. 
Мы называем это «статикой», потому что сервер посылает размещенные файлы в браузер 
«как есть».

Динамический веб-сервер состоит из статического веб-сервера и дополнительного 
программного обеспечения, чаще всего сервера приложения и базы данных. Мы называем 
его «динамическим», потому что сервер приложений изменяет исходные файлы перед 
отправкой в ваш браузер по HTTP.
